## Getting Started

1. Clone the repo

```
https://github.com/Mayur2506/Smart-Menu-Card.git
```
2. Move Into Working Directory
```
cd Smart-Menu-Card
```

3. At the Project root run

```sh
npm install
```

```sh
node server/index.js
```
4. Move into the client folder

```
cd client
```
5. Run The Project using 

```
npm install
```

```
npm start
```
